# Security Policy

## Supported Versions

Only the latest version of `quran-search-engine` is supported for security updates.

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please do NOT open a public issue. Instead, report it privately:

- **Email:** <contact@adelpro.us.kg>
- **GitHub:** You can also use the "Report a security vulnerability" button on the repository's security tab if available.

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Non-Security Bugs

For general bugs or performance issues, please open a regular [Issue](https://github.com/adelpro/quran-search-engine/issues).
